# HorizonScan360

**AI-powered horizon scanning for UK public sector governance**

*Turns legislation, policy and regulatory signals into assignable workflows with full audit trails.*

---

### What We Build

| | |
|---|---|
| 📡 **Signal Ingestion** | Automated monitoring of legislation.gov.uk, bills.parliament.uk, GOV.UK, ONS and more |
| 🤖 **AI Assessment** | LangChain4j-powered relevance, impact and urgency analysis per organisation profile |
| ✅ **Governance Workflows** | Signals converted to tracked, assignable actions with deadlines |
| 🔒 **Audit Trail** | Immutable log of every decision and state change |
| 🏛️ **Multi-Tenant** | Built for NHS Trusts, local authorities and other UK public bodies |
